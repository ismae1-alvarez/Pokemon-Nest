# Ejecutar en desarrollo

1. Clonar el respositorio
2. Ejecutar
```
  pnpm install
```
3. Tener Nest CLI instalado
```
  pnpm i -g @nestjs/cli
```
4. Levantar la bse de datos
```
  docker-compose ip -d
```

## Stack usado 
* MongoDB
* Nestjs
